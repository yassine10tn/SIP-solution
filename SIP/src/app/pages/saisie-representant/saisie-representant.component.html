Gestion des représentants

{{ isPermanent ? 'Représentants permanents' : 'Représentants ponctuels' }}

Le représentant permanent est affecté à une société pour une durée indéterminée, jusqu'à son remplacement.

Le représentant ponctuel est mandaté pour participer à une seule réunion précise. Son pouvoir est valable uniquement pour cette réunion.

La société est obligatoire.
Le matricule est obligatoire.
Le nom et prénom sont obligatoires.
